## 📌 News Trend Analyzer

Проект для сбора и анализа новостных данных по заданным ключевым словам с использованием API.

---

## 🎯 Цель проекта

Разработать Python-приложение, которое:

* собирает новости по заданным темам
* анализирует данные
* выявляет тренды
* сохраняет результаты в удобном формате

---

## 🔧 Используемый API

* NewsAPI
  или
* GNews API

---

## ⚙️ Функциональность

### 1. Сбор данных

Приложение должно:

* принимать ключевое слово (например: `AI`, `crypto`, `marketing`)
* отправлять запрос к API
* получать список новостей за указанный период (например, 7 дней)

---

### 2. Обработка данных

Для каждой новости извлекаются:

* источник (source)
* заголовок (title)
* дата публикации (publishedAt)

---

### 3. Аналитика

#### 📊 Базовая аналитика:

* количество новостей по запросу
* топ источников (по количеству публикаций)
* самые часто встречающиеся слова в заголовках

#### 🔥 Продвинутая аналитика:

* распределение новостей по дням
* анализ заголовков:

  * средняя длина
  * наличие цифр
  * наличие “кликбейтных” элементов (`!`, CAPS)
* выявление популярных тем (частотный анализ)

---

### 4. Вывод результатов

Приложение должно:

#### 📁 Сохранять данные:

* в CSV файл (`news.csv`)
* (опционально) в JSON (`news.json`)

#### 🖥 Выводить в консоль:

```bash
Top sources:
1. TechCrunch — 25 articles
2. Forbes — 18 articles

Top words:
AI, startup, funding, growth
```

---

## 🚀 Запуск проекта

Пример запуска через CLI:

```bash
python main.py --query "AI" --days 7
```
